# Personal Portfolio Website

This is my personal portfolio website built using **HTML, CSS, and JavaScript**. It presents a brief introduction about me, my skills, and my work in a clean and simple layout.

## Features
- Personal introduction section
- Education Achievements & Awards
- Skills and experience overview
- Projects & Articles and Contacts
- Responsive and minimal design


## Technologies Used
- HTML5
- CSS3
- JavaScript

## Deployment
This project is deployed using **Vercel** and connected to **GitHub** for automatic deployments.
